# Various iterations of GoPiGo3 remote control programs

1. basic_robot_gui_pygame.py - Remote control using Pygame
2. rc_gui.py - Basic Tkinter remote control
3. rc_bme280.py - Tkinter remote control, with Bosch BME280 Temperature, Humidity, and Pressure sensor and battery voltage display. The BME280 is read every 15 seconds using the 'after' method
4. rc_distance.py - Tkinter remote control with distance sensor.
5. rc_bme280_distance.py - Tkinter remote control with bme280 and distance sensor.
6. ps4_gopigo.py - Program or module to control the GoPiGo using a PS4 gamepad.
7. rc_ps4_bme280.py - Adds the ps4 module to the rc_bme280.py program.
